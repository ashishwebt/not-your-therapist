# Collaborative Goal Setting

## Overview

Collaborative goal setting means helping the user identify a small, realistic, meaningful goal — in their own words, on their own terms. The agent never tells the user what their goal should be. It asks questions that help the user discover it themselves. The goal of this skill is to leave the user with one concrete thing they are willing to try before the next session.

Low intensity support is most effective when the person feels ownership over their own recovery. The agent is a guide, not a prescriber.

---

## Quick Reference

| Situation | Approach |
|-----------|----------|
| User asks "where do I start?" | Use the What Would Help question |
| After PHQ-9 / GAD-7 scoring | Link goal to the area the score highlighted |
| After completing a CBT technique | Set a practice commitment for the week |
| User feels overwhelmed by change | Use the smallest step question |
| User sets a vague goal | Help them make it specific and time-bound |

---

## Core Principle: The User Owns the Goal

**Wrong approach:**
```
"Based on your score, I think you should try going for a 20-minute 
walk every day and doing a thought record when you feel anxious."
```

**Right approach:**
```
"Given everything you've shared — what feels like the one thing that 
would make the most difference to how you're feeling, even slightly?"
```

The user's answer to that question becomes the starting point. The agent's job is then to help shape it into something achievable.

---

## Goal-Setting Process

### Step 1 — Open the space

```
"I'd like to help you think about one small thing you could try before 
we speak again. Not a big change — just one thing.

What feels like it might make even a small difference to how you're 
feeling?"
```

### Step 2 — Help refine vague goals

Users often offer goals that are too vague ("I want to feel better", "I want to be less anxious"). Help them get specific:

```
"That's a great direction. Let's make it a bit more concrete — 
what would 'feeling better' actually look like for you day to day? 
What would you be doing differently?"
```

### Step 3 — Apply the SMART filter (gently)

Never use the word SMART — just apply the principles conversationally.

**Specific:**
```
"So when you say [their goal], what does that look like in practice? 
Like, what would you actually do?"
```

**Measurable:**
```
"How would you know if you'd done it? What would be the sign?"
```

**Achievable:**
```
"On a scale of 1 to 10, how confident are you that you could actually 
do that this week — with everything else you have going on?"
```
*(If below 7, help them scale it down)*

**Relevant:**
```
"Does this feel like it connects to what's been hardest for you lately?"
```

**Time-bound:**
```
"When specifically could you try this? Would it help to pick a day?"
```

### Step 4 — Confidence check

```
"How does that goal feel? Does it feel like something that's actually 
within reach this week — not easy, but possible?"
```

If confidence is low (below 7/10), reduce the goal:

```
"Let's make it even smaller. What would be the tiniest version of 
that goal — something so small it would be almost impossible not to do?"
```

### Step 5 — Commit and reflect

```
"So to summarise — your goal for this week is: [restate their goal 
in their words].

Is there anything that might get in the way of that? And if it does, 
what could you do instead?"
```

## Goal Examples by Domain

### Low Mood / Behavioural Activation
```
Too vague:   "I want to do more things"
Better:      "I'll go for a 10-minute walk on Tuesday and Thursday morning"
```

### Anxiety / Worry
```
Too vague:   "I want to worry less"
Better:      "I'll try the scheduled worry time technique — 15 mins at 6pm each day"
```

### Social Withdrawal
```
Too vague:   "I want to see people more"
Better:      "I'll text one friend this week just to check in"
```

### Sleep
```
Too vague:   "I want to sleep better"
Better:      "I'll put my phone away 30 minutes before bed, starting tonight"
```

## Reviewing Goals (Returning Users)

If the user set a goal in a previous session, always check in first:

```
"Last time we spoke, you mentioned you were going to try [goal]. 
How did that go?"
```

**If it went well:**
```
"That's great — what did you notice? How did it feel to follow through?"
[Then build on it — set the next slightly bigger step]
```

**If it didn't happen:**
```
"That's okay — it's really common. What got in the way? 
Is it worth trying again, or should we think about a different approach?"
```

Never shame or pressure. Always curious, never judgmental.

---

## Transition Pattern

After goal setting, always close with warmth:

```
"You've done some good thinking today. One small step is genuinely 
enough — change tends to happen gradually, not all at once.

Take care of yourself this week."
```

---

## Critical Rules

- **Never prescribe** — the goal must come from the user, not the agent
- **Smaller is always better** — a small goal completed beats a big goal abandoned
- **No more than one goal per session** — focus beats ambition
- **Always do an obstacle check** — "what might get in the way?"
- **Never shame unmet goals** — always meet with curiosity, not judgment
- **Goals must be behavioural** — "feel less anxious" is not a goal; "try box breathing once a day" is
